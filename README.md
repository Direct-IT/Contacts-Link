Un petit plugin sous GLPI 10.0.20 qui permet d'insérer facilement un contact dans le suivi ou la résolution d'un ticket.

INSTALLATION

1. Télécharger la dernière version du plugin 
2. Dézipper le tout dans votre repertoire plugins/
3. Dans GLPI, allez dans Configuration / Plugins
4. Installer puis activer le plugin Contacts Link
5. C'est fini

UTILISATION
1. Rédiger un suivi ou une solution
3. Taper # suivi de quelques lettres de votre contact (créé dans Gestion > Contacts)
4. Sélectionner votre contact
5. L'insertion est faite avec un lien vers sa fiche quand le suivi ou la solution sera enregistré.
